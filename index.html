UniVST: A Multi-Conditional Adapter for Unified Voice Style Transfer with Style Manipulation

Anonymous

Abstract

Voice Style Transfer (VST) modifies speech to match a reference style (e.g., timbre, emotion) while preserving its content. Most VST approaches pay attention to single style transfer and have made remarkable progress. Recent approaches have attempted to transfer multiple styles by disentangling speech attributes and enabling controllable style conversion. However, these methods typically rely on style-conditioned generation without considering the original speech style during training, leading to limited preservation of source speech attributes.

To address these issues, we formulate style transfer as a style-manipulation task that simply adjusts the style of the source speech instead of generating it from scratch. Therefore, we propose a unified Voice Style Transfer framework UniVST dedicated to transferring both single style and multiple styles. Built on a Siamese-like diffusion model, UniVST consists of a Source Branch for extracting source speech features and a Conversion Branch for generating converted speech. We first employ a Multi-Condition Embedder (MCE) to extract rich style descriptions from natural language and precise timbre from personalized audio prompts. We then propose a Multi-Conditional Adapter (MCA), which employs feature manipulation to extract and modulate style features from the source speech. These modulated style features are subsequently integrated into the conversion process via masked attention, enabling fine-grained style control while preserving the attributes of the source style. During inference, classifier-free guidance improves transfer controllability, and a Large Audio-Language Model (LALM)-based Length Regulation refines speech rate and duration. Experimental results demonstrate that UniVST can achieve natural and accurate style transfer in multiple VST tasks compared to existing methods.

Multi-Style Transfer

Given the following inputs:

  • Text Prompt: A sentence or phrase that describes the intended speaking style.
  • Audio Prompt: An audio sample representing the target speaker identity.
  • Source Speech: The original input speech containing the content to be retained.
  • Generated Speech: The output speech that should reflect the target speaker and desired style.
